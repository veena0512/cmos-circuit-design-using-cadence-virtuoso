# CMOS Circuit Design using Cadence Virtuoso

## Overview
This project demonstrates the design and simulation of CMOS inverter circuits using Cadence Virtuoso.

## Features
- CMOS inverter schematic design
- Transient waveform analysis
- Layout design

## Tools Used
- Cadence Virtuoso
- Cadence genus

## Results
The inverter circuit was successfully simulated and analyzed for switching characteristics and propagation delay.
